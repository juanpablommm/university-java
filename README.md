* **Spring Boot** este projecto de spring nos permite crear aplicaciones java de manera muy simples...

* **Spring Cloud Data Flow** el cual nos permite comunicar de mejor manera aplicaciones que utlizan micro-servicios

* **Servlet Stack** donde podremos crear aplicaciones que utlizan el patron de dise√±o MVC para la parte web, pero tambien aplicaciones de tipo REST para comunicarnos con otro tipos de tecnologias, como para comunicarnos con clientes tipo Angular, etc, este se apoya del contenedor de los Servlets, usa la API de los Servlets, en la cpa de datos utilza tecnologias como JDBC, JPA, NoSQL
